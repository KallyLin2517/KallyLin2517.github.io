C语言·基本概念

2020-03-15 C语言

编写一个简单的C程序

首先，需要先下载一个编译器，小甲鱼的C语言教学视频用的是VC++6.0。一开始我是用这个，不过学校开始上课后就改成Dev-C++和PieTTY。我们学校机房用的是PieTTY（这个必须连学校的网络才能用）,我私下敲代码更喜欢用Dev。

当安装好编译器后，就可以开始打第一段代码。很多人应该都是以“Hello World”开始，我也是，而且总觉得这句话有种特别的浪漫在里边。

#include<stdio.h>

int main(){

printf(“Hello World!\n”);

return 0;

}

就以上这个简单的程序为例，C语言一般分成指令和函数两部分。

“#”后面的就是指令，“main”是函数。函数内包含语句，语句一般需要在末尾加上分号。

printf是输出函数，功能是打印双引号内的内容，printf函数中，双引号内的内容是字符串字面量。

“\n”是换行符，若不使用换行符的话printf不会自己跳转到下一行输出。

注释

  • 单行注释：使用“//”
  • 多行注释：使用“/* */”

示例如下：

printf(“It’s really boring to record this.”)//打印语句

printf(“Yeah, but what can you do?”)

/*打印输出

Yeah, but what can you do?*/

变量的使用

变量是用来存储数据的单元，类型有很多种，如整型，浮点型，字符型，枚举类型等。

声明：

变量使用前需要声明，格式是“变量类型 变量名称”。如：

int boring;

赋值：

赋值有两种，一种是在声明变量时顺带赋值，称为初始化变量；一种是在声明之后赋值，需要注意这个必须先声明后才能赋值。如：

int boring=100;//初始化

或者

int boring;

boring = 100;

读入输出

读入函数有scanf()和getchar()等，前者可以读入任何基本数据，后者是读入单个字符；输出函数有printf()和putchar()等，前者可以输出字符、数字等等，后者输出单个字符。

scanf()使用格式如下：

scanf(“%d”,&really);

scanf(“%c”,&boring)

其中%d代表整型，%c代表字符型，意味着输入的必须是整型/字符型。&后面的是代表输入的数据存入该变量中，&是取址运算符。在调用scanf函数时大多时候要把&加上去，只有某些时候不需要。

代码书写规范

当代码书写途中，我们需要对变量、函数、宏和其他实体进行命名。这些名字称为标识符，C语言中标识符可以含有字母、数字、下划线，但是必须以字母或下划线开头。
