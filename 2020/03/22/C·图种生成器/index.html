C·图种生成器

2020-03-22 C语言

前言

程序教程来自B站小甲鱼《C语言与程序设计》最后关于“文件操作”的几P，这个东西我自己取名图种生成器，其实是我自己的恶趣味。更准确的说法应该是文件合成器才对，因为本质就是把两个文件拼接到一起。

需要具备的知识点

会敲C语言，会用C语言文件操作的几个文件。如果不会的话也没关系，右转小甲鱼教程，你就是下一个图种制造大师。

或者，啥都不会也可以，只要是个活人会调出控制台就行。

能够实现的功能

每个人的电脑里可能会有些不想被人发现的秘密，假如这个秘密藏在一个rar压缩文件里，你可以通过该程序把它和一张图片拼接到一起。这样，平时的后缀名为jpg时，看起来就是普通得不能再普通得图片；但是如果把后缀名改成rar，同样是一个可执行程序，可以解压。

实现

简易版

  1. 首先，因为我的控制台打开后默认位置是C盘的用户>lenovo，我就先把一个1.jpg文件和一个2.rar放到这个文件夹下，这两个文件就是等会要合成的素材。
  2. win+R打开控制台，输入cmd，回车。
  3. 输入copy 1.jpg+2.rar 3.jpg，其中3.jpg是合成后的文件名，输入之后回车。
  4. 好了，刚刚的文件夹就多了一个3.jpg的文件，点开看是图片。但是如果把后缀名改成rar，它就会解压出2.rar里的东西。我为什么要在这里记这些这么傻白甜的操作

进阶版

进阶版就是自己写一个程序了，事实上只要那几个函数会用了，不难的。

代码思路大致如下：

  1. 输入第一个文件的名字，用fopen（）函数，rb方式打开这个文件。（这个文件必须存在）
  2. 用第一步的方法打开第二个文件。
  3. 输入第三个文件的名字，用fopen（）函数，wb方式打开这个文件（实际上是新建）。
  4. 用fgetc（）函数读取第一个文件和第二个文件每一个字符，用fputc()函数将读取到的写入第三个文件中。
  5. 大功告成。所以我为什么要在这里记这些这么傻白甜的操作

代码如下：

#include<stdio.h>
int main(){

FILE *fp,*fp2,*fp3;
char ch,filename[20]={0},filename2[20]={0},filename3[20]={0};    
printf("请输入第一个文件名字：");

scanf("%s",filename);

if(!(fp=fopen(filename,"rb") )){
    printf("找不到指定文件\n"); 
    return;
}
printf("请输入第二个文件名字：");
scanf("%s",filename2);
if(!(fp2=fopen(filename2,"rb") )){
    printf("找不到指定文件\n"); 
    return;
}
printf("请输入要生成的文件名字：");
scanf("%s",filename3); 
fp3=fopen(filename3,"wb");
while(!(feof(fp))){
ch=fgetc(fp);
fputc(ch,fp3);
}
fclose(fp);
while(!(feof(fp2))){
    ch=fgetc(fp2);
    fputc(ch,fp3);
}
printf("已经生成\n"); 
fclose(fp2);
fclose(fp3);
return 0;

}
