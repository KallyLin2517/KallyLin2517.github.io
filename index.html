问，如何快速创建一个自己的网页？

2020-03-12 网页

简易版

3分钟使用github制作一个属于自己的网站（B站视频）

让你使用GitHub无需编程制作出真正精美的网站（B站视频）

过程

注册github -> 创建仓库 -> 下载gridea -> 使用gridea部署

优点

简单易上手，相比用hexo，gridea界面过于清新简洁了。

缺点

截止发稿日，朋友告知这个软件目前只支持github框架，且主题较（很）少。而且因为简洁的界面，肯定会失去很多功能。如果想深入研究这
方面内容的，最好还是考虑hexo。

其他相关资料

gridea文档

进阶版

以下两个教程差不多，可以只选一个看

hexo史上最全搭建教程

手把手教从零开始在GitHub上使用Hexo搭建博客教程

过程

注册github -> 创建仓库 -> 安装git -> 安装Node.js -> 安装hexo -> 用hexo部署

优点

由于博主本人并不了解hexo所以说不出太多优点，但是某朋友认为github page这个渠道中，hexo是很高的等级了主题多，支持的比较多。

缺点

有点难（大佬自动忽略这句话），容易出现各种各样见过或没见过的错误警告。这时候，如果你并不了解这些错误代表什么，请复制黏贴百度一下。

其他相关资料

Hexo文档

总结

只想创建自己的网页，好好写博客的可以考虑gridea。想要更进一步研究，或者觉得gridea太简单毫无挑战性的可以接触hexo。

关于该网站

网站的图标是L，我姓氏的首字母。C·T·H代表的是“courage”，“try”和“hope”，也是我在做这个网页时最大的感触。

本来是打算直接用我名字的拼音缩写做网站名字，不过效果一上来看了下有点怪怪的，果断换掉。

最开始用了gridea做管理工具，可是它主题太少了，完全满足不了我，于是换了hexo。该网页的主题是hexo主题Alpha Dust，第一眼就喜欢上了，因此就拿来用。

网站里有什么

有可爱美丽动人大方善良的网站主人和她的学习笔记。

社交链接

网站底部的社交链接是无用的，因为懒我还没把自己的社交链接放上去（再说我也没那么多社交账号）。本来想直接把图标都屏蔽掉，但是屏蔽之后网站底部空空的，好像不太好看，遂决定保留。
